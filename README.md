# Kareem Radwan — Personal "About me" Website

This repository contains a small static website you can host with GitHub Pages.

Files:
- `index.html` — main page
- `styles.css` — styles
- `resume.pdf` — optional (add your resume)
- `CNAME` — optional (add your custom domain)

Quick setup (recommended: user site):
1. This repository is created as a user site: `kareemkr.github.io`.
2. Clone it locally:
   ```bash
   git clone https://github.com/kareemkr/kareemkr.github.io
   cd kareemkr.github.io
   ```
3. Copy or replace the files above into this folder.
4. Commit and push:
   ```bash
   git add .
   git commit -m "Initial personal site"
   git push origin main
   ```

Visit: https://kareemkr.github.io after a minute — GitHub Pages will publish the site automatically for a user site.

Customization
- Replace the avatar SVG with your image: add `assets/avatar.jpg` and change the markup in `index.html`.
- Update the social links, projects, skills, and contact info.
- To use a custom domain, add a CNAME file containing the domain and set it in repository Pages settings.
